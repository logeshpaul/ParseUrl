Parse Data from URL and Compare Products.

This project is all about parsing and compare product from a third party site.

Run Application

There are two ways to run this application

Simple Way:

  • Download the project
  • Open index.html file in a browser to run the app (As simple as that)

Gulp Way:

  • To speed up development I have used Gulp + Sass + Browser Sync. You can run it using gulp if you want.
  • Make sure you have already installed node and gulp in your machine.
  • Download the project & navigate to the project path in terminal
  • Run 'npm install gulp gulp-sass browser-sync' to install the node modules for the project
  • Just type 'gulp' in terminal to trigger the server!
